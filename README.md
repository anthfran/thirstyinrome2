# Thirsty in Rome

An iOS app for finding drinking fountains (*nasoni*) in Rome.

## About

This is a complete remake of my first ever iOS app, originally built in 2017 while living in Rome. The original was created manually — every line written by hand. This version is an experiment in building with [Claude Code](https://claude.ai/code), Anthropic's AI-powered developer tool.

## Features

- Full-screen map centred on Rome with all known *nasoni* marked
- Marker clustering when zoomed out — tap a cluster to zoom in
- Re-center on Rome button
- Re-center on GPS button with live authorization and signal state

## Built With

- Swift / SwiftUI
- MapKit / CoreLocation
- iOS 17+, Xcode 26
- Claude Code (claude.ai/code)
